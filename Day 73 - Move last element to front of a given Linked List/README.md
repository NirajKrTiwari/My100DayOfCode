Move last element to front of a given Linked List
--------------------------------------------------
TestCase 1:

Input:
Linked list before moving last to front
50 20 15 4 10 
Output:
Linked list after moving last to front
10 50 20 15 4

TestCase 2:

Input:
Linked list before moving last to front
3
Output: 
Linked list after moving last to front
3
