Binary Search Tree Minimum element
-----------------------------------
Given a Binary Search Tree (BST) that may be unbalanced, find the minimum element in it.
example
Input:
        5
      /   \
     3     7
    / \   / \
   2   4 6   8
Output: 2
Input:
        5
      /   \
     3     7
    / \   / \
   2   4 6   8
        / \
       1   9
Output: 1
Input:
        5
      /   \
     3     7
    / \   / \
   2   4 6   8
        / \
       1   9
      /
     0
Output: 0
